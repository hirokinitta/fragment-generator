# 記憶断片生成器 — Fragment Generator

**v0.1.0 フィードバック先行版**

---

## このツールについて

「記憶断片生成器」は、**不思議な世界観・liminal space的なシーン**を自動生成するジェネレーターです。

廃校の体育館、深夜のショッピングモール、どこにも繋がらない廊下——
現実にありそうで、どこかおかしい。
そんな「記憶の断片」のようなシーンをランダムまたは手動パラメータで生成します。

創作のインスピレーション源として、世界観探しのお供として、
あるいは単純に眺めて楽しむ用途としてご活用ください。

---

## インストール方法

1. `Fragment Generator Setup 0.1.0.exe` をダブルクリック
2. **Windowsの警告が出た場合** → 「詳細情報」→「実行」をクリック
   （署名なしのアプリのため警告が出ますが、安全です）
3. インストール先を選択して「インストール」
4. デスクトップまたはスタートメニューから起動

> **動作環境**：Windows 10 / 11（64bit）

---

## 基本的な使い方

### シーン生成

左パネルの **[ GENERATE ]** を押すだけで、シーンが生成されます。

生成されるのは以下の要素の組み合わせです。

- **タイトル** — シーンの断片的なタイトル
- **LOCATION / TIME / LIGHTING** — 場所・時間・照明
- **EMOTION** — その場の感情・雰囲気
- **ANOMALY** — 違和感・異常（非現実度が高いほど出現）
- **SOUND** — 聞こえてくる音
- **COLOR** — 全体のカラートーン
- **COMPOSITION** — 構図（アングル・レイアウト）

---

### RANDOM / MANUAL モード

- **RANDOM** — すべてのパラメータをランダムに決定
- **MANUAL** — スライダーで3つのパラメータを調整

  | パラメータ | 効果 |
  |---|---|
  | NOSTALGIA（懐かしさ） | 学校・団地・商店街など過去の空間が増える |
  | ANXIETY（不安） | 廊下・病院・閉塞空間が増える |
  | UNREALITY（非現実度） | 異常現象（ANOMALY）が出やすくなる |

> **ヒント**：全パラメータを90以上にすると…何かが起きます。

---

### MIXモード

複数生成したシーンから要素を組み合わせて、新しいシーンを作るモードです。

1. **[ GENERATE ]** でシーンを2つ以上生成する
2. 上部タブの **MIX** をクリック
3. 右の履歴パネルからシーンを **ドラッグ＆ドロップ** で A・B に設定
4. 各フィールドの **A / B ボタン** をクリックして採用元を切り替える
5. **SHUFFLE** でランダム切り替え、**COPY** でテキストとしてコピー

---

### 色図鑑（COLOR）

シーンで使われる色パレット40色を、カテゴリ別に一覧表示するコーナーです。
それぞれに世界観のキャプションが付いています。

---

### お知らせ・アップデート

タイトルバー右上の **ベルアイコン** からお知らせを確認できます。
アップデートがある場合はここに通知が届きます。

## データについて

生成したシーンは自動的にローカルに保存されます。

```
保存場所：
C:\Users\[ユーザー名]\AppData\Roaming\fragment-generator\fragments.db
```

- ネット接続は**不要**です（アップデート確認時のみ使用）
- 生成データは外部に送信されません
- アンインストールしてもデータは残ります
  完全削除する場合は上記フォルダを手動で削除してください
